Detailed Description

This library implements EM to estimate the parameters of a GMM distribution from data.

The algorithm starts with 1 mixture. Then: Iterate till required number of mixtures it iterates: -split mixtures

  • Iterate til convergence (EM):
  • Expectation
  • Maximization

There are functions to

  • write GMM to disk
  • read GMM from disk
  • compute prob. of data given the GMM ...
